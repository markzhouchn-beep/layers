# Layers 产品需求文档

> 版本：2.0 | 日期：2026-04-14 | 状态：开发中

---

## 1. 项目概述

**Layers** — 中国创作者 · 全球周边平台

一个 Print-on-Demand 平台，帮助中国独立创作者将原创作品通过外部平台（Gumroad / Etsy / Amazon 等）销往全球，同时通过 Printify 处理生产和物流。创作者只需上传作品，平台负责其余一切。

### 核心价值
- **创作者**：零成本入驻，人民币订阅费，躺着赚钱
- **消费者**：独占中国原创设计，高性价比
- **平台**：订阅费 + 外部平台销售分成 + Printify 差价

### 收入模型
```
收入 = 创作者订阅（CNY）+ 外部平台销售（USD）+ Printify 差价
成本 = Printify 生产成本 + 平台运营
利润 = 收入 - 成本
```

## 2. 三个端的核心定位

### 2.1 创作者端 — `/creator`
**用户**：中国独立插画师、设计师
**语言**：简体中文
**目标**：让创作者零门槛入驻，持续创作，坐等收钱

| 功能 | 说明 |
|------|------|
| 注册/登录 | 邮箱+密码，订阅方案选择 |
| 作品上传 | 图片 + Mockup 生成器 + 产品预览 |
| 订阅管理 | Free/Basic/Pro 三档，CNY 支付 |
| 数据面板 | 总收入、总销量、外部平台汇总 |
| 外部账号 | 绑定 Gumroad / Etsy 等平台 |
| 收入提现 | 版税通过 Stripe Connect 结算 |

### 2.2 消费者端 — `/`
**用户**：全球消费者（北美/欧洲/澳大利亚为主）
**语言**：英文为主，支持中/法切换
**目标**：发现、预览、购买中国原创周边

| 功能 | 说明 |
|------|------|
| 首页 | 精选作品瀑布流、分类 |
| 作品详情 | Mockup 效果图、产品类型选择 |
| 外部购买 | 点击跳转外部平台（Gumroad/Etsy/Amazon） |
| 创作者主页 | 查看某艺术家全部作品 |
| AI 销售更新 | 实时同步外部平台价格/库存变化 |

### 2.3 运营管理端 — `/admin`
**用户**：Layers 平台运营团队
**语言**：简体中文
**目标**：审核作品、管理创作者、监控全局

| 功能 | 说明 |
|------|------|
| 数据大盘 | 收入、订单、创作者、订阅统计 |
| 作品审核 | 批准/拒绝创作者提交的作品 |
| 创作者管理 | 列表、禁用、订阅变更 |
| 订单管理 | 外部平台订单同步状态、Printify 生产状态 |
| 外部平台 | 各平台连接状态、销售数据聚合 |
| 财务管理 | 平台收入、创作者版税结算 |

## 4. 外部平台销售策略

### 4.1 为什么这样设计
- 阿里云服务器无法直接访问 Printify API（被拒）
- 域名 DNS 被劫持，部署独立站有障碍
- 现有电商平台（Gumroad / Etsy / Amazon）已有流量和信任
- AI Agent 可以自动化大部分操作

### 4.2 流程
```
创作者上传作品 → 审核通过
       ↓
平台生成 Mockup → 自动发布到 Gumroad / Etsy / Amazon
       ↓
消费者购买（外部平台付款）
       ↓
AI Agent 轮询检测 → 同步订单数据
       ↓
触发 Printify 生产订单 → 国际物流发货
       ↓
创作者收到版税通知
```

### 4.3 外部平台集成
| 平台 | API 状态 | 优先级 |
|------|---------|--------|
| Gumroad | API 成熟，有认证 | P0 |
| Etsy | API 需要申请 | P1 |
| Amazon Merch | 需注册 | P2 |
| Redbubble | API 封闭 | 待定 |

### 4.4 AI Agent 职责
- 轮询外部平台 API（每15分钟）
- 检测新订单 → 写入数据库 → 触发 Printify
- 同步价格、库存状态变更
- 通知创作者新订单
- 生成销售报告

## 5. 创作者端功能详情

### 5.1 注册流程
1. 填写基本信息（用户名/邮箱/密码/艺术家名）
2. 选择订阅方案（Free/Basic/Pro）
3. 完成注册 → 进入创作者后台

### 5.2 作品上传与 Mockup
- 上传原创图片（JPG/PNG，≤10MB）
- 选择产品模板（T恤/马克杯/海报/画布/手机壳/帆布包）
- 拖拽调整图片位置/缩放/旋转
- 实时预览产品效果图（红色虚线 = 印刷区域）
- 导出 PNG 文件用于外部平台
- 提交审核

### 5.3 数据面板
- 累计总收入（所有外部平台汇总）
- 本月收入
- 总订单数
- 各平台销售排名
- 近30天销售趋势图

### 5.4 外部平台管理
- 绑定平台账号（OAuth 或 API Key）
- 查看各平台店铺链接
- 单作品同步状态

## 6. 消费者端功能详情

### 6.1 首页

## 7. 运营管理端功能详情

### 7.1 数据大盘
- 今日收入（CNY + USD）
- 本月新增创作者
- 待审核作品数
- 活跃外部平台订单
- 趋势图（7天/30天）

### 7.2 作品审核
- 待审核列表（图片 + 标题 + 创作者 + 提交时间）
- 批准 → 自动发布到选定的外部平台
- 拒绝 → 附理由
- 批量操作

### 7.3 创作者管理
- 列表（ID/用户名/邮箱/订阅方案/状态）
- 查看详情、变更订阅
- 禁用/启用账号

### 7.4 订单管理
- 外部平台订单同步列表
- Printify 生产状态（已下单/生产中/已发货）
- 物流追踪

## 8. 技术架构

### 8.1 前端
- React 18 + TypeScript + TailwindCSS v4 + Vite
- 三套独立页面（同一代码库，路由分隔）
- 共享组件库
- i18n：EN / ZH / FR

### 8.2 后端
- Express.js + Node.js 22
- JWT 认证（创作者/管理员）
- PostgreSQL（创作者数据、订单、收入）
- 外部平台 API 轮询服务

### 8.3 AI Agent
- OpenClaw 内置定时任务（cron）
- 每15分钟轮询外部平台
- 订单数据写入 PostgreSQL
- 触发 Printify 生产

### 8.4 文件存储
- 图片：阿里云 OSS
- 备选：本地文件系统
