# Layers 产品需求文档

> 版本：1.0 | 日期：2026-04-14 | 状态：待开发

---

## 1. 项目概述

**Layers** — 中国创作者 · 全球周边平台

帮助中国独立创作者将原创作品通过 Print-on-Demand 模式销往全球 100+ 国家和地区，实现零库存、零物流的被动收入。

### 核心价值主张
- **创作者**：人民币订阅费（低成本入场），美元销售（汇率套利）
- **消费者**：独家的中国创作者原创设计，高性价比定制产品
- **平台**：订阅费 + 汇率差 + 交易抽成

---

## 2. 目标用户

### 2.1 创作者（供给侧）
- 中国独立创作者、插画师、设计师
- 已有 Instagram 账号（有follower基础）
- 渴望变现但无供应链能力

### 2.2 消费者（需求侧）
- Instagram 重度用户，18-35岁
- 追求个性化和独特性
- 主要市场：北美、欧洲、澳大利亚

---

## 3. 商业模式

### 3.1 订阅方案

| 方案 | 年费 | 版税 | 适用 |
|------|------|------|------|
| Free | ¥0 | 8% | 试水期 |
| Basic | ¥299 | 30% | 成长期 |
| Pro | ¥599 | 45% | 成熟期 |

### 3.2 收入流
```
创作者订阅（人民币）→ 平台收入
消费者购买（美元） → Printify 生产 → 国际物流
                       ↓
              平台抽 15% + Printify 成本
                       ↓
              创作者版税通过 Stripe Connect 结算
```

---

## 4. 功能需求

### 4.1 创作者端
- 注册 / 登录（邮箱 + 密码）
- Stripe Connect 收款账户连接
- 上传原创作品（图片 + 标题 + 描述 + 标签）
- 提交审核 → 批准后自动上架 Printify
- 查看销售数据 / 版税收入 / 申请提现

### 4.2 管理员端
- 审核作品（批准 / 拒绝）
- 创作者管理 / 订单管理 / 统计

### 4.3 消费者端
- 浏览作品（首页推荐 / 分类 / 搜索）
- 作品详情（含效果图预览）
- 选择产品类型（12款 curated 产品）
- Stripe 美元结算

---

## 5. MVP 范围（第一阶段 / 2周）

**必须：**
- 创作者注册 + 登录
- 作品上传 + 审核流程
- Stripe Connect 收款（创作者端）
- Curated 12款产品展示
- 消费者购买（Stripe USD）

**可选（后续）：**
- Instagram OAuth
- 自动内容发布
- 订阅支付（人民币）
- OSS 图片存储
