# Layers 前端设计规格

> 版本：1.0 | 日期：2026-04-14

---

## 1. 设计理念

**「墨·境」** — 中国水墨的留白哲学 × 现代国际印刷品质感

- 大面积留白，内容即呼吸
- 黑白墨色为主调，中国红/朱红为点缀
- 动态微妙：不打扰，但让每一次交互都有回应
- 艺术品永远是主角，UI 是背景而非竞争者

---

## 2. 设计系统

### 2.1 色彩

| 角色 | 色值 | 用途 |
|------|------|------|
| 墨黑 | `#1a1a1a` | 主文字、标题 |
| 纸白 | `#fafaf8` | 主背景 |
| 暖灰 | `#f5f4f0` | 卡片、分割区域 |
| 朱砂 | `#c9382a` | 主 CTA、强调（谨慎使用） |
| 烟灰 | `#6b6b6b` | 次要文字 |
| 浅墨 | `#e8e7e3` | 边框、分隔线 |
| 竹青 | `#2d5a47` | 成功状态、艺术家认证 |
| 藤黄 | `#c49a3a` | 提示、促销 |

### 2.2 字体

- **主字体**：Inter（正文、UI）
- **中文回退**：`"PingFang SC", "Noto Sans SC", sans-serif`
- **字重**：300（轻盈）/ 400（正文）/ 500（按钮）/ 600（标题）

### 2.3 间距

- 基准单位：4px
- 常用：8, 12, 16, 24, 32, 48, 64, 96px

### 2.4 圆角

- 小（按钮/输入框）：8px
- 中（卡片）：12px
- 大（特殊卡片）：16px

### 2.5 阴影

```css
/* 微妙纸张阴影 */
--shadow-sm: 0 1px 3px rgba(0,0,0,0.06);
--shadow-md: 0 4px 16px rgba(0,0,0,0.08);
--shadow-lg: 0 12px 40px rgba(0,0,0,0.12);
```

### 2.6 动效

- 过渡时长：200ms（快）/ 350ms（标准）/ 500ms（戏剧）
- 缓动：`cubic-bezier(0.4, 0, 0.2, 1)`
- 悬停：微微上浮 + 阴影加深
- 页面进入：淡入 + 微上移

### 2.7 图标

- **库**：Lucide React
- **风格**：线条图标，stroke-width: 1.5
- **尺寸**：16px（紧凑）/ 20px（标准）/ 24px（强调）

## 4. 组件清单

### 导航栏
- Logo（左）+ 菜单项（中）+ 登录/语言切换（右）
- 滚动后背景从透明变纸白，加底部细线阴影
- 移动端：汉堡菜单 → 全屏抽屉

### 作品卡片
- 图片（aspect-ratio: 4/5）+ 标题 + 艺术家名 + 价格
- 悬停：图片微放大（1.03）+ 阴影加深
- 支持懒加载

### 产品选择器
- 网格展示 12 款 Printify 产品
- 选中状态：朱砂边框

### 底部 CTA
- 墨黑背景 + 纸白文字
- 大字标题 + 邮件输入框 + 按钮

### 步骤条
- 数字圆点 + 连接线
- 完成：竹青色 / 当前：朱砂 / 未达：浅灰

## 5. 技术实现

### 5.1 技术栈
- **框架**：React 18 + TypeScript
- **构建**：Vite
- **样式**：Tailwind CSS v4 + CSS Variables
- **图标**：Lucide React
- **动画**：Framer Motion（可选）/ CSS transitions
- **状态**：React Context + localStorage
- **路由**：React Router v6

### 5.2 目录结构
```
client/
├── src/
│   ├── components/     # UI 组件
│   │   ├── Navbar.tsx
│   │   ├── Footer.tsx
│   │   ├── ArtworkCard.tsx
│   │   ├── ProductPicker.tsx
│   │   └── ...
│   ├── pages/          # 页面
│   │   ├── Home.tsx
│   │   ├── ArtworkDetail.tsx
│   │   ├── ArtistProfile.tsx

│   ├── hooks/          # 自定义 hooks
│   ├── context/        # React Context
│   ├── utils/          # 工具函数
```

### 5.3 响应式断点
- Mobile：< 640px
- Tablet：640px - 1024px
- Desktop：> 1024px
